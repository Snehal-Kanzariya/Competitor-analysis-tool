# Users and Use Cases

## Primary user
The primary user is a founder, product manager, marketer, or analyst who wants
a quick, structured understanding of competitors for a given company or product.

## User skill level
- Understands basic business and products
- Does NOT need to understand AI, prompts, or data sources
- Wants clarity, not raw data dumps

## Supported use cases

### Use case 1: Identify competitors
User provides a company or product name.
Agent returns a list of direct and indirect competitors.

### Use case 2: Compare positioning
User wants to understand how competitors position themselves
(features, pricing approach, target audience).

### Use case 3: High-level strengths and weaknesses
Agent summarizes publicly visible strengths and weaknesses
based on messaging, features, and market presence.

## Unsupported use cases

- Deep financial analysis
- Custom business strategy creation
- Market size estimation
- Investment or acquisition advice
- Real-time competitive monitoring
- Internal or confidential company analysis

## Behavior rules
- The agent must stay within the defined use cases
- If a request falls outside scope, the agent should politely refuse
- The agent should prioritize clarity over completeness
