# Competitor Analysis AI Agent – Project Overview

## Purpose
This project builds an AI agent that can analyze competitors of a given company or product and produce structured insights.

## What this agent is NOT
- Not a chatbot
- Not a general research assistant
- Not a brainstorming tool

## Core idea
Given a company or product, the agent:
1. Identifies competitors
2. Collects public data
3. Analyzes positioning, strengths, weaknesses
4. Outputs a structured competitor report

## Constraints
- Spec-driven only
- No assumptions without data
- Deterministic, repeatable outputs
