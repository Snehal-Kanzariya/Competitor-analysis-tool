# Problem Definition

## The problem we are solving
People want to understand their competitors, but:
- Information is scattered across many sources
- Manual research is slow and inconsistent
- Insights depend too much on who is doing the analysis

This agent solves this by producing a consistent, structured competitor analysis
from publicly available information.

## Input to the agent
- Name of a company, product, or startup
- Optional: industry or market (if known)

## Output from the agent
- A fixed-format competitor analysis report
- Same structure every time
- Based only on available public data

## What success looks like
- Two different runs with the same input produce similar results
- Output is clear enough to compare competitors side by side
- No vague or fluffy insights

## Out of scope (the agent must refuse or avoid)
- Giving business advice or strategy recommendations
- Predicting future revenue, growth, or valuation
- Using private, paid, or insider data
- Making assumptions without sources
- Acting as a general research assistant
