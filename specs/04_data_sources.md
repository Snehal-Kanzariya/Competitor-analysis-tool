# Data Sources

## Purpose
This document defines where the agent is allowed to obtain information
and how that information should be interpreted.

The agent must only use publicly accessible data.

## Allowed data sources

### Primary sources
- Official company websites
- Product landing pages
- Public documentation
- Public blog posts maintained by the company

### Secondary sources
- Public comparison pages
- Public press releases
- Well-known public directories (e.g. startup listings)

All sources must be accessible without login or payment.

## Disallowed data sources

- Paid tools or subscription databases
- Private or leaked information
- User-generated speculation (forums, Reddit, comments)
- Social media opinions
- Review scores without context

## Data handling rules

- Prefer first-party sources over third-party summaries
- If multiple sources conflict, state the uncertainty
- If no reliable data is found, say so explicitly
- Never fill gaps with assumptions

## Transparency requirement

Every insight produced by the agent must be traceable
to at least one allowed public source.
